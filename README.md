#  Rashfire Anime Streaming Platform

A complete anime surfing and binge-watch website with integrated merchandise store.

## Features

-  All-Time Top 10 Anime (including Demon Slayer #1)
-  Current Top 10 Anime (Spring 2026 season)
-  Dedicated Demon Slayer Merchandise Store
-  Responsive Design
-  Docker Support
-  CI/CD with GitHub Actions
